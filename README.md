# 臺灣防災速報

歡迎使用《臺灣防災速報》，這是一個無廣告的防災資訊平台，提供即時地震、颱風、高溫警報、避難所與急救資源資訊。

請先登入 GitHub 開始使用防災服務。

---

## 功能

- 地震速報（中央氣象署 CAP XML）
- 高溫警報（Windows 9.0.0）
- 即時推播（Firebase，支援 Android/iOS）
- 可安裝的 PWA App
- 3D 地圖顯示災情（CesiumJS）
- 顯示避難處所、AED 裝置位置
- 提供 CPR 教學及急救先鋒功能模組
- 整合臺灣防災資訊（地震、颱風、淹水、土石流等）
- 即時空拍與監視器畫面（交通部、水利署等來源）
- 無廣告介面
